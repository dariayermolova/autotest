Политика запрещенных слов  

Здесь Вы можете добавить список запрещенных слов, разделяя их запятыми. Слово должно содержать минимум 3 символа. Возможно использовать звездочку (*) как маску символа. Обратите внимание, слово «р*мка» будет влиять на слово «рюмка» и также на слово «рамка».

IP ваших серверов, для прямых вызовов нашего API
